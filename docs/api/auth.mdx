# 1. Auth
**v0 — System Token Only**

Для **MVP** авторизации используется **один тип токена** — системный `access_token` с полным доступом ко всему API. Пользовательских аккаунтов и ролей нет.

Сейчас токен можно получить только в Telegram-чате команды. В будущем возможна доработка процесса авторизации
с генерацией токенов с настроенными правами доступа.
